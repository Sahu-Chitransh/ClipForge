Help & FAQs

Quick fixes for common issues

Why does the backend show as offline?

Make sure the backend is running at `http://127.0.0.1:8000` and that the frontend is opened on the same machine.

Download start nahi ho raha?

Pehle URL add karke metadata load hone do. Phir `Download to PC` ya item ke `Download This` button se request bhejo. Agar playlist hai to `Load Playlist` ke baad bulk options apply karo.

4K/1080p option missing hai?

Woh resolution sirf tab dikhega jab metadata us video ke liye support show kare. Low-resolution videos ke liye dropdown auto-filter hota hai.

Audio me thumbnail kaise aata hai?

Audio downloads MP3 me convert hote hain aur thumbnail usi file ke metadata me embed hota hai. Zip file nahi banती.

Playlist items par same settings kaise lagayein?

Playlist tab me `Playlist Mod Menu` use karo. Format aur quality dropdown se select karte hi woh sab loaded links par apply ho jayega.
